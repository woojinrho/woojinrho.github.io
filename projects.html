Verilog RTL RISC-V 2024 – 2025

RISC-V RV32I Processor

A full implementation of the RISC-V RV32I instruction set architecture in Verilog. Phase 1 is a complete single-cycle processor with all base integer instructions. Phase 2 introduces a 5-stage pipeline with hazard detection and forwarding. Phase 3 adds a custom matrix-multiply hardware accelerator coupled via memory-mapped I/O.
